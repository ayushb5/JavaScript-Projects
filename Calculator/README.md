🔢 Calculator

A simple Calculator using JavaScript that can perform basic calculations.

This project demonstrates the practical use of:

- DOM Manipulation
- Event handling
- Arrow Function
- eval() function
- String methods

Built using HTML, CSS, Bootstrap 5, and JavaScript.

🚀 Features

- ✅ Basic Calculations - Addition, Substraction, Multiplication, Division
- ✅ Show live result
- ✅ All clear function
- ✅ Backspace function
